# Heading 1

    Markup : # Heading 1 #
    -or-
    Markup : ======== (below H1 text)

## Heading 2

    Markup : ## Heading 2 ##
    -or-
    Markup : ----------- (below H2 text)

    Markup : ### Heading 3 ###

Common Text

    Markup : Common Text

_Emphasized Text_

    Markup : _Emphasized Text_
    -or-
    Markup : *Emphasized Text*

    Markup : ***Strong Emphasized Text***
